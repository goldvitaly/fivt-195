Основная функция сортировки - sort.cpp. Интерфейс: ./sort input_file_name [temp_file_name]
Генерация файла заданного размера с случайными числами: ./gen input_file_name n [randseed]
Формат файла: первые восемь байт - размер файла, далее по четыре байта сами числа. Если я ничего не путаю, все в Big Endian.
readgen.cpp - человекочитаемый вывод бинарного файла

myio.h - реализация бинарного и строкового ввода-вывода
sort_algo.cpp - вспомогательные алгоритмы, используемые в сортировке
